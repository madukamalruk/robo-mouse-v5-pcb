## 🛠️ පියවර 1: Altium එකට Project එක දාගැනීම සහ Footprints හැදීම

අපි මුල් ෆයිල් එක හැදුවේ වෙනත් Software එකකින් (JITX) නිසා, Altium එකට ඒක දැම්මම සමහර කෑලි වල 3D හැඩතල (Footprints) අමුතුවට පේන්න පුළුවන්. ඒක හදාගන්න එක තමයි පළවෙනි රාජකාරිය.

### 1.1 Altium එකට Import කිරීම
1. Altium එක Open කරලා, **File → Import Wizard** යන්න.
2. ඒකෙන් **"KiCad Design Files"** තෝරලා, අර මම එවපු `kicad.rar` එක අස්සේ තියෙන KiCad ෆයිල් එක තෝරන්න.
3. Next කරගෙන ගිහින් Finish කරන්න. දැන් ඔයාට කෑලි ටිකයි, බෝඩ් එකේ හැඩයයි පෙනෙයි.

### 1.2 Footprints මාරු කිරීම (Footprint Manager)
අර අමුතු හැඩතල අයින් කරලා Standard Altium හැඩතල දාමු.
1. උඩ මෙනු එකෙන් **Tools → Footprint Manager** යන්න.
2. වම් පැත්තේ ලිස්ට් එකෙන් කෑලි තෝරලා, දකුණු පැත්තෙන් මේ Standard Footprints ටික Assign කරන්න:
   * **Resistors & Capacitors (R සහ C):** `0603` (හරි `0402` හරි) කියන Standard SMD footprint එක දාන්න. (0603 දැම්මොත් පාස්සන්න ලේසියි).
   * **STM32F405 (U1 - MCU):** මේකට **`LQFP-64`** කියන Footprint එක දාන්න (කකුල් 64ක් තියෙන සමචතුරස්‍ර හැඩයක්).
   * **DRV8833 (Motor Driver):** මේකට **`HTSSOP-16`** කියන Footprint එක දාන්න (යටින් තඹ පාඩ් එකක් තියෙන්න ඕන).
   * **ICM-42688 (IMU):** මේකට **`LGA-14`** කියන එක දාන්න.
   * **ToF Sensors 4 (VL53L0X මොඩියුල්):** මේවා කඩෙන් ගන්න මොඩියුල් නිසා බෝඩ් එකට පාස්සන්නේ නෑ. ඒ නිසා මේවට දාන්න ඕන **`HDR1X6`** (1x6 Pin Header, Pitch 2.54mm) කියන Footprint එක.
   * **OLED Display එක:** මේකට දාන්න ඕන **`HDR1X4`** (1x4 Pin Header, Pitch 2.54mm).
3. ඔක්කොම මාරු කළාට පස්සේ **Accept Changes (Create ECO)** ඔබලා Execute කරන්න.

## 📚 පියවර 2: Layers සහ Rules හැදීම (වඩාත්ම වැදගත්!)

### 2.1 තට්ටු 4ක් (4-Layers) සැකසීම
මේ රොබෝ එකේ මෝටර් 2ක් තියෙන නිසා ලොකු ඝෝෂාවක් (Electrical Noise) එනවා. ඒකෙන් බේරෙන්න බෝඩ් එක තට්ටු 4කට හදන්නම ඕන.
1. Altium එකේ **Design → Layer Stack Manager** යන්න.
2. එතන Layers 4ක් හදන්න:
   * **Top Layer:** Signal / Routing (රතු පාට)
   * **Inner Layer 1:** Plane (මෙතන "GND" නෙට් එක Assign කරන්න. මේක සම්පූර්ණයෙන්ම තඹ වලින් පිරුණු පලිහක්!)
   * **Inner Layer 2:** Plane (මෙතන "3.3V" නෙට් එක Assign කරන්න).
   * **Bottom Layer:** Signal / Routing (නිල් පාට)
3. Save කරලා වහන්න.

### 2.2 Design Rules සැකසීම (Tracks වල මහත)
Altium එකට අපි කියන්න ඕන මොන ඉර කොච්චර මහතට අඳින්න ඕනද කියලා. **Design → Rules** යන්න.
**Routing → Width** යටතේ මේ නීති ටික හදන්න:
1. **මෝටර් Tracks (AOUT1, BOUT1, VM):** මෝටර් වලට ලොකු කරන්ට් එකක් යනවා. ඒ නිසා මේවායේ Width එක **Min: 1.0mm, Preferred: 1.5mm, Max: 2.0mm** දෙන්න.
2. **Power Tracks (5V, 3.3V, VBAT):** මේවාට Width එක **Min: 0.5mm, Preferred: 1.0mm, Max: 1.5mm** දෙන්න.
3. **සාමාන්‍ය Signal Tracks (I2C, SPI, GPIO):** මේවා තුනී වුණාට කමක් නෑ. **Min: 0.15mm, Preferred: 0.25mm, Max: 0.5mm** දෙන්න.

**Clearance (ඉඩ):** Electrical → Clearance වලට ගිහින් එක ඉරක් සහ තව ඉරක් අතර අවම පරතරය **0.127mm** (5 mils) දෙන්න.

## 🧩 පියවර 3: කෑලි Place කිරීම (Component Placement)

මේක තමයි ගේම් එකේ අමාරුම කොටස. මම කියන තැන් වලම හරියටම කෑලි ටික තියන්න. Altium එකේ කෑල්ලක් උඩ Click කරලා Spacebar එක එබුවම ඒක 90° හැරෙනවා. Properties Panel එකෙන් ඕනෑම Angle එකක් දෙන්නත් පුළුවන්.

### 1. IMU (ICM-42688-P) - මුලින්ම තියන්න ඕන කෑල්ල!
* **තැන:** බෝඩ් එකේ හරියටම ජ්‍යාමිතික මැද (Geometric Center). රෝද දෙකට කපලා තියෙන හිඩැස් දෙක (Wheel cutouts) හරහා ඉරක් ඇන්දොත්, අන්න ඒ ඉර හරි මැද තමයි මේක තියෙන්න ඕන.
* **ඇයි?:** රොබෝ කැරකෙද්දී හරියටම හැරෙන කෝණය මනින්න මේක මැදම තියෙන්න ඕන.

### 2. මෝටර් ඩ්‍රයිවර් (DRV8833) සහ Connectors
* **තැන:** රෝද වලට කපපු හිඩැස් ගාවින්ම තියන්න. හැබැයි IMU එකෙන් අවම 10mm වත් ඈතට කරන්න.

### 3. මොළය (STM32F405 MCU)
* **තැන:** IMU එකට ටිකක් ඉස්සරහින් හරි පස්සෙන් හරි මැදට වෙන්න තියන්න.
* **අනිවාර්යයයි (VCAP):** මේකේ Pin 31 සහ 48 වලට එන 2.2µF Capacitors දෙක MCU එකේ කකුල් වල ගෑවෙන තරම් ළඟින් තියන්න. **මේක ළඟින් තිබ්බේ නැත්නම් රොබෝ On වෙන්නේ නෑ!**
* **Decoupling (100nF):** හැම VDD පින් එකක් ගාවම 100nF Capacitor එකක් කකුලේ ගෑවෙන තරම් ළඟින් තියන්න. (මේවා වතුර ටැංකි වගේ, ක්ෂණිකව කරන්ට් එක දෙනවා).

### 4. බිත්ති බලන ToF Sensors 4 (1x6 Headers)
මේවා තියන්න ඕන ඉස්සරහා අර්ධ රවුමේ දාරේ දිගේ (Edge එකේ). කෑල්ල තෝරලා Properties වලින් Rotation Angle එක මේ විදියට දෙන්න:
* **Front Left & Front Right:** ඉස්සරහා බිත්තිය බලන්න මේ දෙක බෝඩ් එකේ ඉස්සරහා අර්ධ රවුමේ ඉදිරියටම වෙන්න තියන්න. Angle එක 0° (කෙලින්ම ඉස්සරහට) හරි, පොඩ්ඩක් එළියට 15° හරි හැරිලා තියෙන්න ඕන.
* **Side Left & Side Right:** පැත්තේ බිත්ති බලන්න මේ දෙක රවුමේ දෙපැත්තට වෙන්න තියන්න. Angle එක හරියටම 45° හරි 90° හරි එළියට හැරිලා තියෙන්න ඕන.

### 5. IR Sensors
* ToF සෙන්සර්ස් අතරින් ඉස්සරහා දාරයේම තියන්න. මේවත් 45° කෝණයකට එළියට හැරිලා තියෙන්න ඕන (බිත්තියට වදින ආලෝකය පරාවර්තනය වෙලා එන්න ඕන නිසා).

### 6. OLED Display එක (1x4 Header) සහ I2C Pull-Up Resistors
* **තැන:** බෝඩ් එකේ පිටිපස්සේ තියෙන පැතලි දාරය (Back flat edge) ගාව.
* **හරස් අතට තියන්න:** OLED එකේ පින් 4 (Header එක) බෝඩ් එකේ පිටිපස්සේ තිරස් අතට (Horizontal) තියන්න. එතකොට Screen එක බැටරිය උඩින් හරි, බෝඩ් එකෙන් පිටිපස්සට පැනලා හරි තියෙයි. මේක තියද්දී පිටිපස්සේ Buttons වලට ඇඟිල්ල තියන්න ඉඩ තියෙනවද කියලා බලන්න.
* **Pull-Up Resistors:** I2C ලයින් (SDA, SCL) වලට 4.7kΩ (හරි 10kΩ) Resistors දෙකක් ඕන. මේ දෙක MCU එක ගාවින්ම තියන්න. (එතකොට Signal එක drop වෙන්නේ නෑ).

## ✏️ පියවර 4: Tracks අඳිමු! (Interactive Routing)

දැන් කෑලි ටික තියලා ඉවරයි. දැන් තියෙන්නේ Altium එකේ **`Ctrl + W`** ඔබලා (හරි උඩින් තියෙන Interactive Routing අයිකන් එක ඔබලා) ඉරි ටික අඳින්නයි.

### රන්වන් නීති 3ක් (Golden Rules):
1. **මෝටර් Tracks යටින් විතරයි (Bottom Layer - Blue):** DRV8833 එකේ ඉඳන් මෝටර් එකට යන Tracks අනිවාර්යයෙන්ම Layer 4 (Bottom) එකෙන් විතරක් යවන්න. Vias (බෝඩ් එක හිල් කරලා යටට යන පාරවල්) පාවිච්චි කරලා යටට අරන් යන්න. Altium එකේ Track එක අඳින ගමන් කීබෝඩ් එකේ **`*`** (තරුව) එබුවම ඉබේම Via එකක් වැටිලා අනිත් Layer එකට මාරු වෙනවා! මෝටර් Tracks 1.5mm මහතට අඳින්න අමතක කරන්න එපා.
2. **Crystal එක යටින් මුකුත් යවන්න එපා:** MCU එක ගාව තියෙන Crystal එක යටින් වෙන කිසිම Track එකක් යවන්න එපා. 
3. **90° කෝණ එපා:** Tracks අඳිද්දී කවදාවත් T අකුර වගේ 90° කෝණ වලට හරවන්න එපා. හැමවෙලේම 45° කෝණ වලින් (Y අකුර වගේ) ලස්සනට හරවන්න. Altium එකේ ඉබේම 45° හැරෙනවා.

### GND Polygons (බිම පිරවීම)
ඔක්කොම ඇඳලා ඉවර වුණාට පස්සේ, Top Layer එකේ ඉතුරු වෙලා තියෙන හිඩැස් සේරම GND (තඹ) වලින් පුරවන්න ඕන.
* **Place → Polygon Pour** යන්න.
* Layer එක Top තෝරන්න. Net එක GND තෝරන්න.
* මුළු බෝඩ් එකම කවර් වෙන්න කොටුවක් අඳින්න. එතකොට ඉබේම හිඩැස් ඔක්කොම තඹ වලින් පිරෙනවා (Noise එක අඩු වෙනවා). ඒක Bottom Layer එකටත් කරන්න.
